# Max-Win-Win 知识库摘要

## 核心原则

1. **双赢思维**：创造增量价值，让客户获得比付出更多的回报
2. **信任优先**：B2B销售的核心是信任建立
3. **价值而非价格**：销售解决方案，而非产品本身
4. **长期主义**：建立持续关系，而非一次性交易

## 销售本质

1. 销售是无处不在的价值交换过程
2. 销售是信任的建立过程
3. 销售是价值的创造过程
4. 销售是有规律可循的科学过程
5. 销售是内在素养与外在技能的统一

## 信任模型（CARE）

- **C - Competence（专业能力）**：做行业专家，每次会面准备2-3个对客户有启发的数据
- **A - Authenticity（保持真实）**：拒绝销售脚本，承认局限，言出必行
- **R - Reliability（保持可靠）**：一致性产生复利，宁可提前不要迟到
- **E - Empathy（展现共情）**：提供心理空气，站在客户的战壕里

## 信任修复四步法

1. 迅速道歉：不推诿责任
2. 解释原因：提供事实，不作借口
3. 实质补救：给出具体补偿方案
4. 透明跟进：保持高频进度更新

## 价值四个维度

1. **功能价值**：产品基础特性
2. **经济价值**：财务指标改善，如节省成本
3. **战略价值**：支持客户长期目标
4. **个人价值**：决策者职业成功、风险降低

## 价值创造三步法

1. 发现价值：通过深度提问发掘隐性痛点
2. 定制价值：解决方案 = 产品 + 服务 + 洞见
3. 展示价值：量化ROI，证明高价产品长期廉价

## 销售四大规律

1. **漏斗规律**：关注每个阶段转化率
2. **量质乘积规律**：成功 = 活动数量 × 活动质量
3. **周期规律**：B2B周期3-6个月
4. **价值交换规律**：每次互动必须提供前置价值

## 销售类型

1. **交易型**：短周期，价格为主要变量
2. **顾问型**：需求复杂，诊断先行
3. **战略级B2B**：长周期6-18个月，多决策中心

## 心态与素养

- 动机纯粹：从"赚佣金"转变为"解决业务难题"
- 深度好奇心：听逻辑而非听需求
- 长期主义：正直是效率最高的销售技巧

## Naval 智慧

- 专属知识：无法被培训替代的深度经验
- 杠杆效应：代码、媒体、人力、资本
- 复利思维：长期关系和知识的积累
- 勇气与体谅平衡：追求双赢，敢于放弃

## 高效能人士七个习惯

- **习惯一**：积极主动 - 为结果负责
- **习惯二**：以终为始 - 设定价值北极星
- **习惯三**：要事第一 - 聚焦第二象限
- **习惯四**：双赢思维 - 把蛋糕做大
- **习惯五**：知彼解己 - 诊断先于处方
- **习惯六**：统合综效 - 创造性协作
- **习惯七**：不断更新 - 持续迭代

## 微博销售洞察（精选）

- 书面沟通的"致谢"模板：正式感谢客户的沟通机会
- 方案排版：页眉、页脚、统一字体、适当段间距
- 书面礼貌：体现专业素养和对客户的尊重
- 优雅的报价：不止报价格，包含封面、致谢、目录、正文、案例、报价
